페이스북 피드 가져오기

사전에 페이스북 로그인 과정이 필요합니다. python의 facebook-sdk 모듈을 활용하여 GraphAPI에 Posts를 요청합니다. 현재 본 앱은 개발 중인 앱이며 본 앱의 개발자가 아니면 해당 API를 이용할 수 없습니다. API 호출 성공 시 사용자 타임라인의 피드 내용들이 담긴 String 배열을 반환하며 전체공개된 게시물만 내용을 가져올 수 있습니다. 에러 시 에러 메시지가 반환됩니다.
